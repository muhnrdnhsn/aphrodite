## FRONT-END
1. Go to root dir
2. Rename ".env.example" to ".env"
3. Run script "npm install"
4. Run script "npm run start-fe"

## BACK-END
1. Go to root dir
2. Go to "backend" dir
3. Rename ".env.example" to ".env"
4. Run script "npm install"
5. Run script "npm run start-be"
